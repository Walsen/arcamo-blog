# Ingeniería de Despliegue

## ¿Qué es la Ingeniería de Despliegue?

Relacionada a la definición de "Despliegue de Software"(*) que se define como "todas las actividades que permiten a un sistema de software estar disponible para su uso". 

Conocida también como Ingeniería de Producción, del anterior texto se desprende la definición "es el arte y/o la técnica de aplicar un conjunto de conocimientos, procesos y habilidades para poner en marcha un sistema de software analizando sus estados, requerimientos, etc."

## Definiciones

Algunas definiciones utiles para entender los procesos y las herramientas relacionadas.

### Proceso

(* General) Sucesión de actos o acciones realizados con cierto orden, que se dirigen a un punto o finalidad, así como también al conjunto de fenómenos activos y organizados en el tiempo. 
(* Informática) Un proceso es una serie de operaciones lógicas y aritméticas ejecutadas por el computador para gestionar datos suministrados y obtener resultados determinados.
(* Economía/Negocios) Secuencia de actividades humanas, que transforman un conjunto específico de insumos en uno de rendimientos. 
(* Industria) Conjunto de transformaciones que se realizan sobre materia prima hasta obtener el producto final terminado.

### Patrón

Serie de variables constantes, identificables dentro de un conjunto mayor de datos.

### Patrón de diseño

Un patrón de diseño es una solución general rehusable para un problema común que ocurre en el contexto del diseño de software.
